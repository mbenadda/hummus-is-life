Coding exercise

Implement a task manager using Angular.JS 1.X. At a minimum, users should be able to add new tasks using the form, which then appear in the list. Other features are optional, although deleting tasks can be an interesting feature to implement.

You can use whatever angular constructs you want to implement the basic functionality in the existing page. Feel free to install other libraries aswell.

Once you're done, please use this as an opportunity to show us what you're good at, the kind of things you want to work on.

Possible improvements could include an additional feature, a better UX, a focus on the internal data structures, some client-server interaction logic, unit tests, a mobile-first experience, offline storage, or something else entirely.
